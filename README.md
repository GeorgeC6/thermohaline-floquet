# Floquet Analysis of Thermohaline Convection

MATLAB code for computing Floquet growth rates of instability in a periodically time-varying background shear flow.

## References

Kochnev, Kalin, "Stability Analysis of Thermohaline Convection With a Time-Varying Shear Flow Using the Lyapunov Method" (2025). Honors Scholar Theses. 1149.

Radko, T. (2019). Thermohaline instability in a time-varying shear flow. *Journal of Physical Oceanography*, 49, 2379–2392.
